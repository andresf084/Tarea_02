## ¿Qué es HTML?

HTML son las siglas de:

-Hyper
-Text
-Markup
-Language

### Hyper Text
Signofica hipertexto y hace refenrencia auntexto que enñlaza con otros conténidos.

### Markup
Significa marca o etiqueta.

### language
Su significado es lenguaje
