## ¿Qué es HTML?

HTML son las siglas de:

- **Hyper**
- **Text**
- **Markup**
- **Language**

### Hyper Text
Significa hipertexto y hace referencia a un texto que enlaza con otros conténidos.

### Markup
Significa marca o etiqueta.

### language
Su significado es lenguaje.
